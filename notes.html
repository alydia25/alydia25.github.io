Can read and make changes to object's properties, make an onject perfomr a task via method, define a procedure that runs when a particular event happens to the object

Refer to a property by using: object.property

Some properties are objects

Lists properties similar to dictionary in Python

Useful Properties: active element, body, childElementCount, children, forms, head, images, lastModified, links, location, title, URL

Can specify element by id, tag name, class name, or selector

Can get elements by class name

To get child, parent.childNodes, for first child, parent.firstElementChild, for last, parent.lastChild, for parent, child.parentNode

To add an element, use document.createElement(elementName), add a new element as a child, such as parent.append(child)

An array is a type of vairable that needs to be deaclared before using, such as const arrayName = new Array();

To populate the array, data can be stored as follows: arrayName[index] = value;

an array can be populated and initialized at the same time by typing values seperated by commas into the () of new Array(); when initializing

Can also use a loop to populate or other iterations, such as forEach()
